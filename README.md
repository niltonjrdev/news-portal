# 📰 TechNews — Technology News Portal

A multi-section technology news portal built with semantic HTML and a utility-first CSS approach using CSS Grid as the primary layout system.

## ✨ Features

- 🗞️ Featured news section with asymmetric grid layout (hero card + 2×2 card grid)
- 📋 Weekly top stories sidebar with image and text cards
- 🤖 Dedicated AI highlights section with article cards (image + text layout)
- 📌 Sidebar with ads slot and "You might have missed" section
- 🏷️ Content category tags reused across all sections
- 🔤 Utility CSS classes for typography scale (`text-sm`, `text-xl`, `text-2xl`)
- 🗂️ Modular CSS architecture with a utility layer (`utility.css`)
- 🧩 Semantic HTML — `<main>`, `<section>`, `<article>`, `<aside>`, `<figure>`, `<figcaption>`

## 🛠️ Tech Stack

- HTML5
- CSS3 (CSS Grid, utility classes — global, header, section, utility, index)
