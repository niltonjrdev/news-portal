# 📰 TechNews — Portal de Notícias de Tecnologia

Portal de notícias de tecnologia com múltiplas seções, construído com HTML semântico e uma abordagem CSS utilitária usando CSS Grid como sistema principal de layout.

## ✨ Funcionalidades

- 🗞️ Seção de destaques com layout de grid assimétrico (card hero + grid 2×2)
- 📋 Sidebar de mais lidas da semana com cards de imagem e texto
- 🤖 Seção dedicada a destaques de IA com cards de artigo (layout imagem + texto)
- 📌 Aside com slot de anúncio e seção "Viu isso aqui?"
- 🏷️ Tags de categoria de conteúdo reutilizadas em todas as seções
- 🔤 Classes CSS utilitárias para escala tipográfica (`text-sm`, `text-xl`, `text-2xl`)
- 🗂️ Arquitetura CSS modular com camada utilitária (`utility.css`)
- 🧩 HTML semântico — `<main>`, `<section>`, `<article>`, `<aside>`, `<figure>`, `<figcaption>`

## 🛠️ Tecnologias

- HTML5
- CSS3 (CSS Grid, classes utilitárias — global, header, section, utility, index)
